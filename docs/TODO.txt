====================

(*)
Bei einem Stream, der so aussieht:
```
[Client:] ...  "report": "321"}
[Server:] HTTP/1.1 200 OK
```
Ist: Die folgende Suche (1) ist kein Match:
```
cdata:"\x22321\x22}" then sdata:"HTTP"
```
Die folgende Suche ist ein Match:
```
cdata:"\x22321\x22}" sdata:"HTTP"
```
Soll: Suche (1) ist ein Match
